# 箱物シール台紙計算ツール

箱物（外箱・中箱）のシール台紙の必要枚数を計算するシンプルな Web ツールです。  
スマートフォンでの現場利用を想定した、1 カラムのカード UI で作られています。

## 機能

- 箱の総数から、必要なシール台紙枚数を計算
- 外箱シールは **1 枚固定**
- 中箱シールは **2 / 8 / 10 個入り**に対応
- シール台紙の綴り枚数は **9 / 10 / 12 枚綴り**に対応
- 台紙枚数は **綴り枚数を分母にした分数表示**（切り上げなし）

例：

- 100 箱 / 外箱 1 枚 / 12 枚綴り  
  → `8 枚 と 4/12 枚（= 約 8.333 枚）`
- 100 箱 / 中箱 2 枚 / 10 枚綴り  
  → `20 枚`

## 初期値

画面を開いたときのデフォルト値：

- 箱の総数：`100`
- 外箱シール：
  - 箱あたり：`1` 枚（固定）
  - 台紙綴り：`12` 枚綴り
- 中箱シール：
  - 箱あたり：`2` 枚（2 個入り）
  - 台紙綴り：`10` 枚綴り

## 使い方

1. 「箱の総数」にパレット上の箱数を入力
2. 外箱シールを使う場合は綴り枚数を選択  
   （使わない場合は「なし」を選択）
3. 中箱の入り数と綴り枚数を選択
4. 「計算する」ボタンを押すと、  
   外箱用・中箱用それぞれの **台紙枚数（整数＋分数＋小数）** が表示されます。

## GitHub Pages での公開方法

1. このリポジトリを `masayamisono/LabelCounter` として GitHub に配置
2. リポジトリの **Settings → Pages** を開く
3. Source を `Deploy from a branch` に設定
4. Branch を `main`（または `master`） / `/ (root)` に設定
5. 保存すると、数分後に `https://masayamisono.github.io/LabelCounter/` で公開されます
